HTML5 / CSS3

Transformations

Transformations de base

Scale

scale(x,y) où "x" et "y" sont des multiplicateurs


.block{
    transform:scale(3,1.3);
}

ScaleX

scaleX(x) où "x" est un multiplicateur


.block{
    transform:scaleX(3);
}

ScaleY

scaleX(y) où "y" est un multiplicateur


.block{
    transform:scaleY(1.3);
}

Translate

translate(x,y) où "x" et "y" sont des distances (px, %, em, ...)


.block{
    transform:translate(200px,50px);
}

TranslateX

translateX(x) où "x" est une distance (px, %, em, ...)


.block{
    transform:translateX(200px);
}

TranslateY

translateY(y) où "y" est une distance (px, %, em, ...)


.block{
    transform:translateY(50px);
}

Rotate

rotate(angle) où "angle" est un angle (deg, rad)


.block{
    transform:rotate(70deg);
}

Skew

skew(angleX,angleY)
où "angleX" et "angleY" sont des angles (deg, rad)

NE PAS UTILISER SKEW

SkewX

skewX(angleX) où "angleX" est un angle (deg, rad)


.block{
    transform:skewX(-50deg);
}

SkewY

skewY(angleY) où "angleY" est un angle (deg, rad)


.block{
    transform:skewY(-50deg);
}

Centre de la transformation

Il est possible de changer le centre de la transformation à l'aide de transform-origin


.block{
    transform-origin:0 0;
    transform:rotate(-70deg);
}

Centre de la transformation


.block{
    transform-origin:50% 50%;
    transform:rotate(-70deg);
}

Centre de la transformation


.block{
    transform-origin:100% 100%;
    transform:rotate(-70deg);
}

}

Backface

L'arrière d'une face n'est pas visible de base
Pour la rendre visible, il faut utiliser backface-visibility ("visible" ou "hidden")


.block-1{
    backface-visibility:hidden;
}
.block-2{
    backface-visibility:visible;
}

Perspective

Il est possible de changer l'impacte de la perspective. Cela ne concerne que les déformations 3D (évident)

Cette modification s'applique sur le container.


.container{
    transform:perspective(800px)
}

Perspective

Il est possible de propager la perspective de manière à obtenir un environnement global avec une même perspective en utilisant transform-style:preserve-3d;

Cette modification doit être appliquée sur chaque objet concerné.


.block{
    transform-style:preserve-3d;
}

Matrices

Il est possible d'appliquer des matrices de transformation.

Mais c'est un trop gros morceau pour ce cours.

En savoir plus pour les matheux et les curieux.


.block{
    transform:matrix(1.06, 1.84, 0.54, 2.8, 466px, 482px);
}
